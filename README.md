
# Ecommerce Spring Boot Backend Project

This is a backend ecommerce project developed using Spring Boot.

## 🚀 Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 Database
- REST API

## 📌 Features
- Add Product
- View Products
- Update Product
- Delete Product
- Search Product

## ▶️ How to Run
1. Clone the project
2. Open in IntelliJ / Eclipse
3. Run the main class
4. Open browser:
   http://localhost:8080
